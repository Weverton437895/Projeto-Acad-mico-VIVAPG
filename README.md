
# Projeto-Acadêmico-VIVAPG

Sistema web desenvolvido para auxiliar moradores, turistas e novos residentes a encontrarem os melhores bairros de Praia Grande-SP com base em indicadores urbanos, custo de vida, segurança, lazer e qualidade de vida.

---

## Sobre o Projeto

O VivaPG é uma plataforma acadêmica criada com o objetivo de reunir informações importantes sobre os bairros da cidade de Praia Grande-SP, permitindo que usuários comparem regiões e encontrem locais mais adequados de acordo com suas necessidades.

O sistema busca transformar dados urbanos em informações acessíveis e intuitivas, ajudando pessoas que desejam:

- Morar em Praia Grande
- Conhecer melhor os bairros da cidade
- Avaliar segurança e qualidade de vida
- Encontrar regiões com melhor infraestrutura
- Comparar indicadores urbanos

---

## Funcionalidades

- Listagem de bairros
- Exibição de indicadores urbanos
- Informações de segurança
- Dados de lazer e infraestrutura
- Comparação entre bairros
- API REST para gerenciamento dos dados
- Interface web integrada ao backend
