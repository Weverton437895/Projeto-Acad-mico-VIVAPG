# Projeto-Acadêmico-VIVAPG

Sistema web desenvolvido para auxiliar moradores, turistas e novos residentes a encontrarem os melhores bairros de Praia Grande-SP com base em indicadores urbanos, custo de vida, segurança, lazer e qualidade de vida.

---

## Sobre o Projeto

O VivaPG é uma plataforma acadêmica criada com o objetivo de reunir informações relevantes sobre os bairros da cidade de Praia Grande-SP, permitindo que usuários comparem regiões e encontrem locais mais adequados de acordo com suas necessidades.

O sistema busca transformar dados urbanos em informações acessíveis e intuitivas, ajudando pessoas que desejam:

- Morar em Praia Grande
- Conhecer melhor os bairros da cidade
- Avaliar qualidade de vida
- Encontrar regiões com melhor infraestrutura
- Comparar indicadores urbanos

---

## Aviso Importante

> Este projeto possui finalidade exclusivamente acadêmica.

Os índices e informações exibidos na plataforma são **estimativas simuladas para fins educacionais**, não representando dados oficiais da prefeitura, órgãos públicos ou instituições de segurança.

O projeto foi desenvolvido apenas para estudo e prática de desenvolvimento de software, banco de dados, APIs e integração entre frontend e backend.

---

## Funcionalidades

- Listagem de bairros
- Exibição de indicadores urbanos
- Visualização dos bairros em mapa
- Informações de infraestrutura e qualidade de vida
- Comparação entre bairros
- API REST para gerenciamento dos dados
- Interface web integrada ao backend

---

## Uso de API e Mapas

O projeto utiliza a API **Nominatim (OpenStreetMap)** para obter as coordenadas geográficas dos bairros de Praia Grande-SP.

Essas coordenadas são utilizadas para:

- Localizar bairros no mapa
- Destacar regiões visualmente
- Melhorar a navegação do usuário
- Exibir informações geográficas de forma interativa

Importante destacar que a API é utilizada apenas para geolocalização e renderização dos bairros no mapa.
